# Task Manager 
A terminal-based to-do list manager to add, update, delete and view your daily tasks.

---

## 1. Description
- Language: Python
- Input: Tasks and menu choices (1-5)
- Output: Updated task list after each operation
- Storage: In-memory list (resets on exit)

---

## 2. How It Works
1. User enters how many tasks to start with
2. A menu appears with 5 options
3. Choose 1 to add, 2 to update, 3 to delete, 4 to view, 5 to exit
4. Loop continues until user selects exit
5. Prints "Thank you for using this to-do list."

---

## 3. Input / Output
| Operation | Input | Output |
|---|---|---|
| Add task | "buy milk" | Task added to list |
| Update task | "buy milk" → "buy eggs" | List updated |
| Delete task | "buy eggs" | Task removed from list |
| View tasks | 4 | Prints full task list |
| Exit | 5 | "Thank you for using this to-do list." |
